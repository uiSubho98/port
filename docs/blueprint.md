# **App Name**: Code Canvas

## Core Features:

- Project Showcase: Display projects with details (title, description, technologies, GitHub link) in a visually engaging manner.
- Skill Visualizer: Use a visually appealing graph to show proficiency in various technologies.
- Experience Timeline: Interactive timeline showcasing professional experience, roles, and key achievements.
- Interactive Bio: A dynamically updated bio that subtly infuses a unique design spin on the person
- Contact Form: A sleek contact form integrated with email for inquiries.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) to represent innovation and technical skills.
- Background color: Dark charcoal (#222831) to make the design elements pop.
- Accent color: Neon Pink (#FF69B4) for interactive elements and highlights to add a funky touch.
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel; suitable for headlines; use 'Inter' for body.
- Subtle parallax scrolling effects for different sections of the page.
- Interactive transitions between projects in the showcase with animated cards sliding in/out.
- Funky animated background textures or patterns that react to mouse movement or scrolling.
